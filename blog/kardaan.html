Kardaan

Coded Using: HTML5, CSS3, Vue.js, Bootstrap

  • 2019-Ongoing
  • Dashboard

I have been working on Kardaan since 2019 for Codematics. My job was to create the Front-End of the dashboard that will be utilized to manage the Kardaan iOS and Android apps.

In the initial meeting, the project manager, the backend developer, and I sat down together to discuss all the functionality that was needed in the dashboard. Then during a meeting, I explained all the functionality to the UI/UX designer. My work started when the full design was provided to me.

I used Vue.js and Bootstrap to design the Front-End and worked alongside the backend developer to integrate the various APIs. This large-scale project required the integration of various npm packages: vue-multiselect, vue-progress, vue-datepicker, vue2-timepicker, sweetalert2, lightbox, moment.js.

After finishing the project, I did rigorous testing of each component before the dashboard was officially launched for use. Now, if any new feature needs to be added, I design and deliver it on an on-going basic. I also provide site maintenance when needed.
